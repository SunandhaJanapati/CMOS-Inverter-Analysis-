# CMOS Inverter using LTspice

## 📌 Description

This project demonstrates the design and simulation of a CMOS inverter using LTspice.

## ⚙️ Components

* PMOS transistor
* NMOS transistor
* Pulse input source
* DC supply (Vdd = 5V)

## 📊 Results

* Propagation Delay ≈ 13 ps
* Rise Time ≈ 321 ps
* Fall Time ≈ 320 ps
* Power Consumption ≈ 20 µW

## 📸 Output Waveforms

For those refer the output images.

## 🚀 How to Run

1. Open `CMOS.asc` in LTspice
2. Click Run
3. Observe waveforms and results
